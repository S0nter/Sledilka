1) Добавить в начало файла название_настройки = значение
2) Продублировать информацию из начала файла в default_settings, где имя ключа совпадает с именем переменной
3) Добавить в Settings элемент, отвечающий за настройку
4) Задать параметры изменения интерфейса исходя из значения переменной настройки в Settings.update_interface()
5) Задать способ сохранения переменной исходя из интерфейса в sett_save

Если тип переменной не встроеный, то прописать индивидуальные параметры в sett_upd(); dataload - readsett()
